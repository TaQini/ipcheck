# IPcheck
----
```
PPAP ~ CheckIP
I have an ip, I have to check.
(Eh~) Check-IP!
I have to check, check an ip.
(Eh~) IP-Check!
Check-IP~IP-Check(Eh~)
IP-Check~Check-IP!
IP-Check~Check-IP!
```
----
##用法
    $ git clone https://github.com/TaQini/ipcheck.git
    $ cd ./ipcheck
    $ chmod u+x ./checkip.sh 
    $ ./checkip.sh
##注意
    1.确保能ping通ip.cn
    2.需要安装python,tracerouter
    3.目前只能在Linux下运行
    4.使用过程中会创建list文件，请确保您有权限在当前目录下创建文件
